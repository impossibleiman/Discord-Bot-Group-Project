Purge

Description
Deletes a chosen number of recent messages from the current channel for quick moderation cleanup.
Permissions
⚙ Manage Messages
Arguments
/purge amountRequired amount of messages to delete. 1-100Allowed range for the amount argument.

Reaction Roles

reactionrole
Description
Creates an interactive setup flow that posts role buttons in a chosen channel so members can self-assign roles without staff involvement.
Permissions
⚙ Manage Roles
Arguments
/reactionrole channelDestination text channel for the button panel. roleRole assigned when a member clicks the button. emojiEmoji shown on the role button.
Actions
select channelPick the destination channel for the setup wizard. type messageEnter the embed or panel message content. select roleChoose the role the button will grant. enter labelSet the text shown on the button. add emojiAdd the emoji shown alongside the label.

Event System

event
Description
Creates and manages temporary event voice channels. Channels are automatically archived or removed after the event concludes.
Permissions
⚙ Manage Channels
Arguments
/event actionChoose start, append, or end. categorySelect the voice channel category. countSet how many voice channels to create. limitOptional user limit for appended channels.
Actions
startCreate the initial event channels. appendAdd more event voice channels. endDelete event channels and stop the event.
