Overview

The Minecraft Society Bot is a collaborative project developed at Manchester Metropolitan University (MMU). It is designed to automate moderation, streamline onboarding, and enhance community interaction within the Minecraft Society Discord server.

The system includes verification, invite tracking, role management, event tools, a ticket system, and an AI chat bot to support a growing community of students.

Welcome & Invite Tracking

Automatically fires when a new member joins. Sends a welcome embed to #test-welcome showing who invited them, which invite code was used, and how old their Discord account is.

Reaction Roles

reactionrole
Description
Creates interactive role-selection buttons in a specified channel. Members click to self-assign roles without staff involvement.
Permissions
⚙ Manage Roles
Usage
/reactionrole [channel] [role] [emoji]

Event System

event
Description
Creates and manages temporary event channels. Channels are automatically archived or removed after the event concludes.
Permissions
⚙ Manage Channels
Usage
/event create [name] [date]

Ticket System

ticket
Description
Opens a private support ticket channel for the member. Staff can view and respond to tickets, and close them once resolved.
Permissions
⚙ Manage Channels
Usage
/ticket
