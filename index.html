Reaction Role Command
Usage
/reactionrole
Description
Creates an interactive setup flow that posts role buttons in a chosen channel so members can self-assign roles without staff involvement.
Permissions
⚙ Manage Roles
Arguments
channelRequiredDestination text channel for the button panel. roleRequiredRole assigned when a member clicks the button. emojiRequiredEmoji shown on the role button.
Actions
select channelRequiredPick the destination channel for the setup wizard. type messageRequiredEnter the embed or panel message content. select roleRequiredChoose the role the button will grant. enter labelRequiredSet the text shown on the button. add emojiAdd the emoji shown alongside the label.
Event Command
Usage
/event
Description
Creates and manages temporary event voice channels. Channels are automatically archived or removed after the event concludes.
Permissions
⚙ Manage Channels
Arguments
actionRequiredChoose start, append, or end. categoryRequiredSelect the voice channel category. countRequiredSet how many voice channels to create. limitOptional user limit for appended channels.
Actions
startCreate the initial event channels. appendAdd more event voice channels. endDelete event channels and stop the event.
